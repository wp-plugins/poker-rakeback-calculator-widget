== Installation ==

1. Upload the entire 'rakeback-calculator' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now you can find Poker Rakeback Calculator Widget in your 'Widgets' panel
